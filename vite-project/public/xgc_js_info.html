Info about Euclid Set and Partition

  • This test program only knows the first 1,000 primes: 2 through 7,919. So 62,710,561 (7,9192) is the largest integer this program can manage
  • An "undefined" Euclid Set means that the following is not true: 0 < Residue < Modulus and Residue prime to Modulus
  • An "undefined" Partition means that the current Euclid Set is not as rich as needed, so you should try to test with a greater number of terms
  • A "false" Partition means that the following is not true: Modulus * First element of the current Euclid Set <= Number <= Modulus * Last element of the current Euclid Set and Modulus divides Number
